#  TickIt - Task Management App

TickIt is a full-stack web application that allows users to register, log in, and manage their daily tasks efficiently.  
It features secure JWT-based authentication, a dynamic dashboard with CRUD functionality, and a clean, user-friendly interface.

---

##  Tech Stack

###  Frontend
- React.js 
- React-Bootstrap for UI
- Axios for API requests
- React Router for navigation
- React Toastify for notifications

###  Backend
- Node.js with Express.js
- MongoDB (Mongoose)
- JWT Authentication
- bcrypt for password hashing
- CORS and cookie-based session handling

---

##  Core Features

###  Authentication
- User Signup / Login / Logout using JWT
- Passwords securely hashed using bcrypt
- Protected routes with authentication middleware

###  Dashboard
- View logged-in user details
- Create, Read, Update, and Delete tasks
- Mark tasks as completed
- Search and filter tasks
- Responsive, modern UI
